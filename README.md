## Usage

Follow along at home with at least stack version 1.7. Each pull request under
[pulls](https://github.com/swift-nav/shake-before-make/pulls) is runnable with:

    ./Shakefile.hs
    
To see more verbose activity:

    ./Shakefile.hs -VV
